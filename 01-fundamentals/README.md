# 01-fundamentals

> Core C# practice focused on syntax, logic, and early object-oriented programming.

This section covers the groundwork behind the rest of the repository: language basics, control flow, methods, classes, constructors, properties, collections, and debugging.

## Areas of practice

- C# syntax and control flow
- methods and parameters
- classes, objects, and constructors
- properties and data handling
- collections and basic problem-solving
- debugging and step-by-step reasoning

## Context
The work here is intentionally foundational. Its role is to support later project work rather than stand on its own as portfolio material.
